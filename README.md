# 포병 계산 프로그램

M109A6 레인지 테이블(`rangeTables/`)을 읽어 저각/고각 포각, ETA, 장약을 자동으로 계산하는 간단한 GUI 도구입니다. 타격지점과 사수의 고도 차이를 고려한 보정도 함께 적용합니다.

## 실행 방법

```bash
python app.py
```

## 사용 방법
1. **나의 고도**, **타격 지점 고도**, **거리(미터)**를 입력합니다.
2. `계산` 버튼을 누르면 고도 차이를 반영한 저각/고각 포각, ETA, 선택된 장약이 각각 최대 3가지 방법으로 표시됩니다.
3. `업데이트 확인` 버튼은 `update_manifest.json`을 읽어 최신 버전을 안내합니다.

## 데이터
- `rangeTables/` 폴더의 `M109A6_rangeTable_low_*.csv`, `M109A6_rangeTable_high_*.csv`를 자동으로 불러와 거리에 맞는 장약을 선택합니다.
- `version.txt`에 현재 프로그램 버전이 기록되며, `update_manifest.json`으로 업데이트 정보를 제공합니다.
