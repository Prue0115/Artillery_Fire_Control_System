## C# WPF 포트
`AFCS.Wpf` 폴더에 기본 계산 로직을 사용하는 WPF 버전이 추가되었습니다. Visual Studio에서 열어 실행하면 `rangeTables` 폴더의 CSV 데이터를 사용해 고각/저각별로 장약을 선택하고 밀·ETA 값을 계산할 수 있습니다.
